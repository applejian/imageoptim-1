Preferências gerais

Preservar permissões, atributos e links para arquivos

Em vez de apagar o arquivo antigo e escrever um novo, irá apenas substituir o conteúdo do arquivo anterior. Isto assegura que todos os links, aliases, proprietários, permissões e atributos adicionais do arquivo são mantidos.

É ligeiramente mais rápido não preservar as permissões.

Criar cópia de segurança do arquivo original antes de salvar

Irá criar uma cópia do ficheiro com ~ adicionado ao nome. Isto torna a optimização um pouco mais lenta e acumula cópias de segurança no seu disco rígido.

Remover todos os pedaços opcionais

Mantenha ativo se estiver optimizando para a Web. Desative se tiver PNGs especiais com metadados adicionais.

Remover marcadores EXIF dos arquivos

Remove metadados da imagem, como o nome do programa ou modelo de câmera que criou o arquivos.

jpegoptim

JPEGOptim

Ativado

Se marcado, o ImageOptim irá tentar otimizar arquivos JPEG.

Qualidade máxima

Se definido como 100%, a optimização será sem perdas.

Qualquer valor abaixo de 100% causará redução de qualidade nos arquivos, se parecerem ter sido salvos com uma definição de qualidade máxima mais elevada.

pngout

PNGOUT

Ativado

Se marcado, o ImageOptim irá tentar otimizar arquivos PNG utilizando o PNGOUT.

O PNGOUT não é distribuído com o ImageOptim. É uma excelente ferramenta, mas o autor do PNGOUT proíbe a sua redistribuição, por isso terá de baixá-lo manualmente.

Tipo de otimização

O PNGOUT é muito eficaz mas também muito lento. Utilize o nível mais baixo se for impaciente.

Interromper se demorar muito tempo a executar

O PNGOUT poderá necessitar de muito tempo para optimizar imagens muito grandes. Esta opção irá interromper discretamente o PNGOUT após 1 minuto.

advpng

AdvPNG

Ativado

Se marcado, o ImageOptim irá tentar otimizar arquivos PNG utilizando a ferramenta AdvPNG.

Nível de optimização
O AdvPNG é relativamente rápido, por isso pode utilizar o nível máximo.
optipng

OptiPNG

Ativado

Se marcado, o ImageOptim irá tentar otimizar arquivos PNG utilizando a ferramenta OptiPNG.

Nível de optimização

O número de diferentes combinações de definições que serão testadas. Raramente fará sentido testá-las todas.

Tornar imagens entrelaçada (progressivas)

As imagens entrelaçadas têm melhor aparência quando estão parcialmente carregadas, mas normalmente o entrelaçamento torna a compressão menos eficaz .

Esta caixa de verificação tem três estados: se assinalada, irá forçar as imagens a tornarem-se entrelaçadas. Se no estado intermédio, irá manter os arquivos inalterados em termos de entrelaçamento. Se desassinalado, irá removar entrelaçamento dos arquivos.

jpegrescan

JpegRescan

Jpegrescan is a script based on jpegtran. It uses creates JPEG files with unusual number of progressive scans to achieve better compression, but such files may not be compatible with all browsers.

gifsicle

Gifsicle

Ativado

Se marcado, o ImageOptim irá tentar otimizar arquivos GIF utilizando a ferramenta Gifsicle.

Arquivos GIF serão alterados para entrelaçados ou não entrelaçados automaticamente, o que resultar num ficheiro menor. Comentários no arquivos serão removidos.

Por favor tenha em mente que o Gifsicle funciona melhor com GIFs animados e que GIFs não-animados raramente podem ser optimizados.

Para melhor compressão, deve converter arquivos GIF para PNGs (O ImageOptim não o faz automaticamente).
